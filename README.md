# web-project
## Project 2 - HTML Foundation Page

**Theme:**  
Personal biography and interest in music.

**What I Learned:**  
In this project, I learned how to structure a basic website using only HTML.
I practiced creating headings, paragraphs, lists, images, tables, and navigation
links. I also learned how early websites handled styling without CSS.

**Challenges:**  
The main challenge was applying styling like spacing and colors using only
HTML attributes. It required experimenting and additional research to understand
HTML limitations before CSS is introduced.
